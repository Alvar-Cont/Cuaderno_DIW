# Cuaderno_DIW

Cuaderno que recopila todas las actividades de la asignatura Desarrollo de Interfaces (2º DAW).

## Estructura

- `index.html`: entrada general del cuaderno.
- `global.css`: estilos base compartidos.
- `ud1/`: Planificacion de interfaces web.
- `ud2/`: Uso de Estilos.
- `ud3/`: Implementacion de la accesibilidad en la web.
- `ud4/`: Implantacion de contenidos multimedia.
- `ud5/`: Tailwind.

## Nota

Este repositorio sirve como cuaderno personal de clase y puede incluir ficheros auxiliares (imágenes, audio y vídeo).
